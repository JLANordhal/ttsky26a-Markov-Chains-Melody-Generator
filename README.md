TT: Stochastic Melody Generator using Markov Chains

This project is a hardware-based procedural music generator designed for the Tiny Tapeout 06 silicon competition. Instead of playing a fixed sequence, it uses Mathematical Probability (Markov Chains) and a Pseudo-Random Number Generator (LFSR) to compose infinite, non-repeating melodies in real-time.

How it Works

The design is divided into three main architectural blocks:

Entropy Source (LFSR): A 16-bit Linear Feedback Shift Register generates pseudorandom noise. This "dice roll" determines the next note and its duration based on a probability matrix.

Stochastic Brain (Markov Chain FSM): This is the core of the project. It implements a Finite State Machine where each state represents a musical note (Do, Re, Mi, Sol, La) or Mute. The transitions between notes are governed by transition matrices, ensuring that the generated melody follows musical logic rather than being completely chaotic.

Audio Engine (PWM Generator): Converts the digital note values into a Pulse Width Modulated signal. By connecting a small speaker or headphones to the output pin, the high-frequency pulses are smoothed into audible tones.

Features

Algorithmic Composition: Generates melodies based on state-transition probabilities.

Configurable Tempo: Toggle between two different BPM settings via input pins.

Dual Transition Matrices: Change the "mood" or style of the melody by switching the probability matrix.

Customizable Seed: Use the 4-bit input to seed the LFSR, ensuring a different starting point for the random sequence every time you reset.

Technical Details

Target Clock: 10 MHz (typical Tiny Tapeout clock).

Note Frequency: Generated by counting clock ticks to match standard musical frequencies.

Implementation: Pure Verilog HDL, designed for the Skywater 130nm process.
